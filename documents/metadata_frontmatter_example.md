# Metadata and Filters Example

This document shows how optional frontmatter can improve retrieval and filtering.

You can search normally with semantic search, or narrow the result set with filters such as:

- `category = guides`
- `tags = ["metadata"]`
- `title_contains = "Filters"`

Custom metadata like `status` and `owner` is also preserved for future use.
